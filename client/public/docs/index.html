wxrks Sync documentation

wxrks Sync sends your Webflow site's Collection items, Pages, and Components to wxrks for translation, then writes the finished translations back into Webflow's real locale variants for each language you've enabled. This guide covers how the app is set up and how each part of it works.

How the sync works, in short

Every send — one-time or recurring — creates a project in wxrks containing the selected content's translatable text. wxrks routes that project through whatever workflow you chose (at minimum, Automatic Translation) and delivers the finished text back to this app via webhook. If a workflow has more than one step, Webflow only gets updated once the last step delivers — an in-progress intermediate step never partially overwrites your site.

Nothing is ever sent silently: every real request to wxrks or Webflow happens either because you clicked Send to wxrks yourself, or because a recurring automation you configured fired on its schedule.
